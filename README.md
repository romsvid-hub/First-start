# Roman Sviderskyi

UI/UX Designer based in Lviv, Ukraine. 6+ years in digital product and UX design, focused on B2B SaaS platforms, complex administrative systems, and data-heavy dashboards.

I work on the parts of products that are hard to make legible: large data tables, admin panels, security dashboards, multi-step cross-platform flows. My focus is structure and decision logic, not surface polish.

### What I do
- Product discovery, user research, information architecture
- End-to-end UX/UI for complex B2B systems
- Design systems (tokens, components, dev handoff)
- Accessibility-focused design (WCAG)

### Selected work

- **All Together Jobs** (IREX-funded). Inclusive hiring platform with WCAG compliance. Led user research, information architecture, and the core Jobs Catalog experience.
